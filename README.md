# Java Word Counter
![screenshot](https://github.com/john-fiore/JavaWordCounter/blob/main/jwc_ss.png)\
This is a simple word counter I wrote in Java as a learning project. I started working on this soon after I graduated high school, but got caught up with other personal life stuff, so I'm finishing it now at 2 AM.

This tool has an **Academic Mode** feature, which essentially adds minimum/maximum word limitations for students.

Considering this was a mere learning project and something for a portfolio, I don't really intend on updating this much. However, if you have feedback for me and this likely crappy code or want me to add something, feel free to [email me](mailto:johnfthedev@gmail.com). Also feel free to contribute to it yourself if you have any knowledge on Java.

> [!NOTE] 
> Eclipse IDE's "WindowBuilder" plugin was used to create this app's GUI,
> hence why the code looks the way it does.
